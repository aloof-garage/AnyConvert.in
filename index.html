Capabilities

Everything you need.
Nothing you don't.

Packed with powerful engines under the hood, wrapped in the simplest interface possible.

Universal Conversion

Convert between 100+ file formats across video, audio, images, PDFs, documents, and subtitles — all from one app.

Fast Processing

Powered by FFmpeg, Sharp, and pdfjs-dist — bundled and optimised for your platform so conversion starts instantly.

Simple Interface

Drop files, pick a format, hit convert. Advanced options are there when you need them, hidden when you don't.

Cross Platform

Native apps for Windows (NSIS + MSIX), macOS (DMG + PKG), and Linux (AppImage + DEB). Runs entirely offline.

Flexible Formats

Supports MP4, MKV, MP3, FLAC, JPG, PNG, WEBP, PDF, DOCX, MD, SRT and dozens more — with batch conversion support.

Zero Setup Required

All engines are bundled. No manual installs, no PATH configuration, no admin rights needed. Download and convert.
